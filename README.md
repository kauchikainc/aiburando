# アイブランド帯広 - デリバリーヘルスウェブサイト

帯広エリアに特化した高級デリバリーヘルスサービスのコーポレートサイト

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **フォント**: Playfair Display (見出し) / Noto Sans JP (本文)

## プロジェクト構成

```
aiburando/
├── app/                    # Next.js App Router
│   ├── page.tsx           # 18歳確認ページ (トップ)
│   ├── home/              # ホームページ
│   ├── cast/              # キャスト一覧
│   ├── info/              # 店舗情報
│   ├── system/            # 料金システム
│   ├── privacy/           # プライバシーポリシー
│   └── terms/             # 利用規約
├── components/            # 共通コンポーネント
│   ├── Header.tsx         # ヘッダー
│   └── Footer.tsx         # フッター
├── data/                  # データ管理
│   └── cast.ts            # キャスト情報
└── public/
    └── images/            # 画像素材
        ├── hero/          # ヒーローセクション画像
        ├── cast/          # キャスト写真
        └── gallery/       # ギャラリー画像
```

## 主要機能

### 1. 18歳以上確認ページ
- 法的要件を満たす年齢確認ゲート
- 18歳未満は閲覧不可

### 2. ホームページ
- 大判背景画像を使用したヒーローセクション
- サービス特徴の紹介
- ピックアップキャスト表示
- CTAボタン (電話・Cityheaven)

### 3. キャスト一覧
- カテゴリー別フィルタリング機能
- ホバー時に詳細情報表示
- Cityheavenへの外部リンク

### 4. 店舗情報
- 営業時間・電話番号
- サービスエリア
- 利用の流れ
- 注意事項

### 5. 料金システム
- 5つの時間コース (60分〜180分)
- オプション料金表
- サービス内容詳細
- 割引制度

## デザインコンセプト

### カラーパレット
- **メインカラー**: ピンク系 (`primary-500`: #ec4899)
- **ベースカラー**: ホワイト
- **アクセント**: イエロー (人気表示)

### UX心理学の適用
- **視覚的階層**: 重要な情報を目立たせる配置
- **アンカー効果**: 具体的な数値で期待を固定
- **ナッジ効果**: 行動を促すCTA配置
- **美的ユーザビリティ効果**: 滑らかなアニメーション
- **段階的開示**: フィルタリングで情報を整理
- **損失回避**: 限定感・緊急性の演出

## 開発環境のセットアップ

### 必要要件
- Node.js 18.x以上
- npm

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

開発サーバーは `http://localhost:3000` で起動します。

### ビルド

```bash
# 本番用ビルド
npm run build

# 本番サーバーの起動
npm start
```
